# 班级电脑控制助手 - 开发者交接文档

## 项目概述

面向教室场景的 Windows 桌面管控工具，帮助教师管理和控制班级电脑的使用。采用 WPF + .NET 10 开发，单窗口多面板架构。

- **当前版本**: 1.0.3
- **目标框架**: net10.0-windows10.0.19041.0
- **UI 框架**: WPF + ModernWpfUI 0.9.6
- **依赖包**: System.Management 10.0.0（WMI 查询）
- **同时引用**: Windows Forms（NotifyIcon 托盘图标、FolderBrowserDialog）

## 核心架构

### 1. 主窗口 (MainWindow)

**单窗口多面板**设计，通过 `Visibility` 切换显示不同功能区域：

| 面板变量 | 功能 |
|----------|------|
| `HomePanel` | 首页仪表盘（系统信息、电池、快捷操作） |
| `AppUsagePanel` | 应用限制管理 |
| `OrganizerPanel` | 桌面整理 |
| `SettingsPanel` | 系统设置 |

**关键机制**:
- `_isInBackground` 标志 + `OptimizeForBackground()`/`RestoreFromBackground()`：窗口隐藏时暂停定时器、恢复时重启
- `_isExiting` 标志：区分"最小化到托盘"和"真正退出"，`OnClosing` 中据此决定是否 `e.Cancel = true`
- `Window_Closing` 事件处理器仅在 `_isExiting = true` 时执行资源清理

### 2. 密码保护体系

- **日志密码** (`_logClearPassword`)：用于保护退出程序、清除日志、恢复默认设置、访问应用限制面板、临时启用被限制应用
- **加密存储**：密码通过 `EncryptPassword`/`DecryptPassword` 加密后保存到 `%LocalAppData%/ClassroomControl/log_password.json`
- **万能密码**：`GetMasterPassword()` 方法通过 XOR 编码在运行时解码，二进制中无明文
- **统一验证**：所有密码验证点使用 `VerifyPassword(input)` 方法，同时检查用户密码和万能密码

### 3. 系统托盘 (TrayIconService)

- 使用 WinForms `NotifyIcon` 实现（WPF 原生不支持托盘图标）
- 右键菜单：打开主界面、关机、重启、退出
- 退出菜单项调用 `_mainWindow.ExitApplication()`，触发密码验证流程
- `MinimizeToTray()` 隐藏窗口并显示气泡提示

### 4. 应用限制 (AppUsageService)

- **监控机制**：`System.Timers.Timer`（2 秒间隔）扫描进程列表
- **匹配规则**：按进程名或路径匹配，支持时间段限制
- **临时放行**：`AllowAppTemporarily()` 设置过期时间，到期自动恢复限制
- **事件通知**：`AppBlockedRequest` 事件触发密码对话框，验证通过后放行 15 分钟
- 进程遍历后调用 `process.Dispose()` 释放资源

### 5. 桌面整理 (DesktopOrganizerService)

- **整理逻辑**：将源文件夹（默认桌面）的文件/文件夹移动到目标文件夹
- **排除规则**：跳过 `.lnk` 快捷方式、用户指定的排除文件夹和扩展名
- **文件名冲突**：自动追加时间戳（`yyyyMMdd_HHmmss`）
- **定时执行**：`DispatcherTimer`（30 秒间隔）检查是否到达设定时间，`_lastExecutedTime` 防止同一分钟重复执行
- **配置持久化**：`%AppData%/ClassroomControl/desktop_organizer_settings.json`

### 6. 定时任务 (DailyTaskService)

- **任务类型**：关机、重启、运行程序、执行命令
- **执行机制**：`DispatcherTimer`（30 秒间隔）检查任务时间
- **防重复**：`LastExecutedDate` 记录上次执行日期，每天只执行一次
- **提醒流程**：到达提醒时间 → 触发 `TaskExecuting` 事件 → 弹出 `ShutdownReminderWindow` → 倒计时结束后执行
- **配置持久化**：`%AppData%/ClassroomControl/scheduled_tasks.json`

### 7. 定时关机调度 (SchedulerService)

- 独立于 DailyTaskService 的简单定时关机功能
- `DispatcherTimer`（1 秒间隔）检查剩余时间
- `_hasReminded` 标志确保提醒只触发一次
- 到达时间后调用 `shutdown.exe` 执行

### 8. 更新服务 (UpdateService)

- **检查更新**：调用 GitHub Releases API (`/repos/{owner}/{repo}/releases/latest`)
- **镜像下载**：`AppConstants.ProxyMirrors` 数组中的代理地址随机选择，用于绕过 GitHub 下载限制
- **版本比较**：支持 `v1.0.3` 和 `1.0.3` 格式，逐段数值比较
- **下载安装**：后台下载 `.exe` 安装包，带进度条，下载完成后启动安装程序

### 9. USB 检测 (UsbDetectionService)

- **设备列表**：WMI 查询 `Win32_LogicalDisk WHERE DriveType=2`
- **设备名称**：多层 WMI 关联查询（DiskDrive → Partition → LogicalDisk）
- **弹出功能**：先尝试 WMI `Dismount`，失败则关闭占用进程后重试，还有 ShellExecute 备选方案
- **实时监控**：`ManagementEventWatcher` 监听 USB 插入/移除事件
- **重要**：所有 `ManagementObjectSearcher` 和 `ManagementObjectCollection` 必须使用 `using` 释放

### 10. 活动日志 (ActivityLogService)

- **存储位置**：`%AppData%/ClassroomControl/activity_log.json`
- **最大条目**：200 条，超出自动裁剪
- **Load() 方法**：调用前会 `Clear()` 防止重复加载

| # | Bug | 原因 | 修复方式 |

| 1 | 密码验证失败 | `LoadLogPassword` 未解密 | 添加 `DecryptPassword` 调用 |

| 7 | 最小化时服务被 dispose | `Window_Closing` 无条件执行资源清理 | 添加 `_isExiting` 检查 |
| 8 | 关机提醒重复弹出 | `ShutdownImminent` 在 1 分钟窗口内每秒触发 | 添加 `_hasReminded` 标志 |
| 9 | 桌面整理重复执行 | 30 秒定时器在同一分钟内触发两次 | 添加 `_lastExecutedTime` 记录 |
| 10 | WMI 资源泄漏 | `ManagementObjectSearcher`/`Collection` 未 dispose | 全部改为 `using` 声明 |
| 11 | 日志重复加载 | `Load()` 未清空集合 | 添加 `Logs.Clear()` |
| 12 | 日志文件路径不确定 | 使用相对路径 | 改为 `%AppData%` 绝对路径 |

## 开发注意事项

### WPF 对话框规范
- 所有 `ShowDialog()` 打开的窗口，使用 `DialogResult = true/false` 关闭，**不要**使用 `Close()`
- 密码输入框应支持 Enter 键提交：`passwordBox.KeyDown += (s, args) => { if (args.Key == Key.Enter) okButton.RaiseEvent(...); };`

### WMI 资源管理
- `ManagementObjectSearcher` 和 `ManagementObjectCollection` 必须使用 `using` 释放
- `ManagementObject` 在遍历时也应调用 `Dispose()`

### 定时器选择
- UI 线程操作用 `DispatcherTimer`
- 后台线程操作用 `System.Timers.Timer`
- 防止同一时间窗口重复触发，需记录上次执行时间或使用标志位

### 密码相关
- 新增密码验证点必须使用 `VerifyPassword()` 方法，不要直接比较 `_logClearPassword`
- 密码加密使用 `EncryptPassword`/`DecryptPassword`，存储路径通过 `GetLogPasswordFilePath()` 获取

### 进程操作
- `Process.GetProcesses()` 返回的每个进程使用后必须 `Dispose()`
- `Process.MainModule` 可能因权限不足抛出异常，需 try-catch

### 窗口生命周期
- 关闭按钮 → `OnClosing` → 检查 `_isExiting` → 最小化到托盘或真正退出
- `Window_Closing` 事件只在 `_isExiting = true` 时执行资源清理
- `App.OnExit` 调用 `ForceDisposeTrayIcon()` 确保系统关机时清理托盘图标

### 更新与发布
- 版本号在 `AppConstants.AppVersion` 中维护
- GitHub Release 的 tag 格式支持 `v1.0.3` 或 `1.0.3`
- 下载 URL 通过代理镜像转发，镜像列表在 `AppConstants.ProxyMirrors` 中配置
